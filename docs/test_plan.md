# sansyouo 実機テスト項目

App Store審査提出前に、以下のテスト項目を実機で確認してください。

## 1. 起動・基本動作

- [ ] アプリが正常に起動する
- [ ] 起動後にクラッシュしない
- [ ] アプリアイコンが正しく表示される
- [ ] ハンバーガーメニュー（右上）が表示される
- [ ] メニューからホーム・グラフ・検索・設定に切り替えできる
- [ ] メニュー外タップでメニューが閉じる

## 2. トップページ

- [ ] 名言テキスト（赤太字）が日本語で表示される
- [ ] 言語を英語に切り替えると名言が英語に変わる
- [ ] サンショウウオのイラスト画像が表示される
- [ ] 背景にオレンジ色＋イラスト画像が半透明で表示される
- [ ] 「サンショウウオ飼育管理記録帳」タイトルが表示される
- [ ] サンショウウオ未登録時に「サンショウウオが未登録です」と表示される
- [ ] [+]ボタンで登録画面が表示される
- [ ] サンショウウオ登録後、Pickerに「名前（種類）」形式で表示される
- [ ] Pickerでサンショウウオを選択できる
- [ ] 編集ボタン（鉛筆アイコン）で編集画面が表示される
- [ ] サンショウウオ未選択時は「記録に進む」ボタンが無効（灰色）
- [ ] サンショウウオ選択時は「記録に進む」ボタンが有効（黒丸・赤文字）
- [ ] 「記録に進む」ボタンは200x200の丸型

## 3. サンショウウオ登録

- [ ] 名前の入力欄が表示される
- [ ] 種類の入力欄が表示される
- [ ] 記念日のトグルがデフォルトOFF
- [ ] 記念日トグルONで日付ピッカーが表示される
- [ ] 名前未入力で「保存」ボタンが無効
- [ ] 名前入力後「保存」で登録される
- [ ] 「キャンセル」で画面が閉じる
- [ ] 複数匹の登録ができる

## 4. サンショウウオ編集・削除

- [ ] 編集画面に既存の名前・種類・記念日が表示される
- [ ] 名前を変更して保存できる
- [ ] 種類を変更して保存できる
- [ ] 記念日を追加・変更・削除できる
- [ ] 削除ボタンで確認ダイアログが表示される
- [ ] 削除するとサンショウウオと全記録が削除される
- [ ] 削除後、Pickerから消える

## 5. 飼育記録カレンダー

- [ ] カレンダーが正しく表示される
- [ ] ハンバーガーメニューが右上に表示される
- [ ] 日付の配置が正しい
- [ ] 日曜日が赤色で表示される
- [ ] 土曜日が青色で表示される
- [ ] 祝日が赤色で表示される
- [ ] 「<」ボタンで前月に移動する
- [ ] 「>」ボタンで次月に移動する
- [ ] 左スワイプで次月に移動する
- [ ] 右スワイプで前月に移動する（前の画面に戻らない）
- [ ] カスタム戻るボタンでトップページに戻れる
- [ ] 年の表示をタップしてメニューが表示される
- [ ] 日付をタップすると選択状態になる（オレンジ背景）
- [ ] 記録がある日付に水換え(青)・エサ(緑)・メモ(グレー)のマーカーが表示される
- [ ] カレンダー上部に凡例（青=水換え、緑=エサ、グレー=メモ、ピンク=記念日）が表示される
- [ ] 記念日の日付にハートマークが表示される
- [ ] 記念日の日付を選択すると記念日メッセージが表示される
- [ ] 記録がある日付に件数バッジが表示される
- [ ] 日付選択時にその日の記録サマリーが表示される
- [ ] 右下の編集ボタンが表示される
- [ ] 日付未選択時は編集ボタンが灰色で無効
- [ ] 日付選択後、編集ボタンタップでDayRecordListViewが表示される

- [ ] 費用トグルON/OFF切り替えができる
